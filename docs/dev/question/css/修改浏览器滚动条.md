# 滚动条

## 1、修改浏览器滚动条样式

```css
::-webkit-scrollbar {
  width: 3px;
}
::-webkit-scrollbar-track {
  border-radius: 10px;
  background-color: rgba(0, 0, 0, 0.1);
}
::-webkit-scrollbar-thumb {
  border-radius: 10px;
  -webkit-box-shadow: inset 0 0 6px rgba(0, 0, 0, 0.3);
  background-color: rgba(0, 0, 0, 0.1);
}
```

## 2、隐藏滚动条

```css
::-webkit-scrollbar {
  display: none;
  width: 0;
  height: 0;
  color: transparent;
}
```
